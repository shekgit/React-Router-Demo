# React Router Demo - Professional Business Website

## Project Overview

A modern, production-ready business website built with React Router v7, featuring dynamic routing, nested routes, and
professional animations. This single-page application (SPA) showcases a complete React ecosystem implementation with
Context API for state management and Tailwind CSS for styling.

## Tech Stack

**Frontend:** React 19 | React Router DOM 7 | JavaScript ES6+  
**Build Tool:** Vite | npm  
**Styling:** Tailwind CSS 4 | Custom Animations  
**State Management:** React Context API  
**Deployment:** Vercel
